SearchResult

A SearchResult can be chained with itself using a variety of functions for advanced indexing.

A search index can be an array or a dictionary. If it is an array, it will only check if the element being searched has each key. If it is a dictionary, it will check if the element has both the key and if the key is equal to the value.

Loops over the current search directory and runs the given callback on every element. This is especially useful for implementing your own logic (ex. checking if a value is greater than or less than a certain value.).

The callback function is supplied with an exclude function. By default, calling forEach() will not alter the search results.
